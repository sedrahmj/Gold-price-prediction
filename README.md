# Gold-price-prediction
Predicting gold prices using historical data and LSTM model.
This project predicts future gold prices using historical data (gold,stocks  and doler).  
It analyzes trends and patterns in gold prices and applies machine learning algorithms to forecast future values.  

The project includes:
- Data preprocessing and cleaning
- Feature selection and engineering
- Predictive modeling using algorithms like Linear Regression, Random Forest, or LSTM
- Visualization of historical trends and predicted prices
- Evaluation of model performance using metrics like RMSE and R²

This system can help investors and analysts make informed decisions based on predicted market trends.
